# HTML specific

**WIP**

## `html`

May include HTML elements that are allowed inside the HTML `paragraph`, except `script` and `style`.

```
[<html><b>bold</b>]
```

renders to

```html
<b>bold</b>
```

## `kbd`

Defines keyboard input.

```
[<kbd>input]
```

renders to 

```html
<kbd>input</kbd>
```

## `samp`

Defines sample program output.

```
[<samp>output]
```

renders to 

```html
<samp>output</samp>
```
